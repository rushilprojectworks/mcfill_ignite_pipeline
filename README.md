# McFill Ignite — Podcast-to-Content Pipeline

Automates post-production content for every McFill Ignite podcast episode.

Upload audio → get 5 assets in under 2 minutes.

---

## What it generates

| Asset | Description |
|---|---|
| Full transcript | Word-for-word via OpenAI Whisper |
| Show notes | Structured with topics, timestamps, takeaways |
| 3 key quotes | Extracted + platform recommendation |
| LinkedIn post | Ready-to-publish, brand-voice copy |
| Team email | Internal summary with asset checklist |
